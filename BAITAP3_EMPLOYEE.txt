# Bài Tập 3 - Employee Management (POJO & Model Binding)

## Mục Tiêu
Phân biệt được đối tượng POJO chứa dữ liệu và cách Spring liên kết dữ liệu.

## Các File Đã Tạo

### 1. Employee.kt (POJO Class)
- **Đường dẫn**: `src/main/kotlin/tam/dev/baitapclassroom/model/Employee.kt`
- **Mô tả**: Lớp POJO chứa thông tin nhân viên
- **Thuộc tính**:
  - `id`: Mã định danh nhân viên (Int)
  - `name`: Tên nhân viên (String)
  - `salary`: Lương nhân viên (Double)

### 2. EmployeeController.kt
- **Đường dẫn**: `src/main/kotlin/tam/dev/baitapclassroom/controller/EmployeeController.kt`
- **Mô tả**: Controller xử lý các request liên quan đến nhân viên
- **Endpoints**:
  - `GET /employee/single`: Hiển thị thông tin 1 nhân viên
  - `GET /employee/list`: Hiển thị danh sách 5 nhân viên

### 3. employee.html
- **Đường dẫn**: `src/main/resources/templates/employee.html`
- **Mô tả**: Giao diện hiển thị thông tin 1 nhân viên

### 4. employee-list.html
- **Đường dẫn**: `src/main/resources/templates/employee-list.html`
- **Mô tả**: Giao diện hiển thị danh sách 5 nhân viên trong bảng

## Cách Chạy Ứng Dụng

1. **Build project**:
   ```bash
   .\gradlew.bat clean build
   ```

2. **Chạy ứng dụng**:
   ```bash
   .\gradlew.bat bootRun
   ```

3. **Truy cập ứng dụng**:
   - Xem nhân viên đơn: http://localhost:8080/employee/single
   - Xem danh sách nhân viên: http://localhost:8080/employee/list

## Các Tính Năng Đã Hoàn Thành

✅ **Yêu cầu 1**: Tạo lớp POJO Employee với các thuộc tính id, name, salary

✅ **Yêu cầu 2**: Tạo đối tượng Employee với dữ liệu cố định trong Controller

✅ **Yêu cầu 3**: Sử dụng Model.addAttribute để đẩy dữ liệu sang View

✅ **Thử thách**: Tạo danh sách 5 nhân viên và hiển thị trong bảng

## Minh Họa Cơ Chế IoC/DI

```kotlin
@Controller
@RequestMapping("/employee")
class EmployeeController {

    @GetMapping("/single")
    fun showSingleEmployee(model: Model): String {
        // Khởi tạo đối tượng Employee
        val employee = Employee(
            id = 1,
            name = "Nguyen Van A",
            salary = 15000000.0
        )

        // Đẩy dữ liệu sang View thông qua Model
        model.addAttribute("employee", employee)

        return "employee"
    }
}
```

## Ghi Chú

- Project sử dụng Kotlin với Spring Boot 4.0.1
- Template engine: Thymeleaf
- Giao diện đơn giản, dễ sử dụng với CSS styling
- Dữ liệu nhân viên được hard-code trong Controller
